## Environment Variables Needed

Create a `.env` file in the root:

```
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_project.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_ANTHROPIC_API_KEY=your_anthropic_key
```

## Key Design Rules (Never Break These)

- **Currency:** Always `₹X,XX,XXX` — never `$`, never plain numbers
- **Numbers:** Lakhs and crores, not millions and billions
- **Colours:** Primary `#1E3A5F` · Accent `#2E86AB` · Success `#1B6B3A`
- **Animations:** XP toast on every gain · badge reveal on every unlock · level-up modal on every level
- **Mobile first:** Design for 375px, then scale up with sm:/md:/lg: breakpoints
- **AI Coach FAB:** Always visible bottom-right (except Landing and Auth pages)
- **Situation Rounds:** Always show the user's actual virtual portfolio snapshot inside the scenario
- **PMS/SIF/AIF:** Always show the real-world minimum investment prominently + educational disclaimer
- **Bridge Rounds:** Cannot be skipped — they gate simulator feature unlocks
- **Leaderboards:** Rankings use normalised metrics — Pro users cannot pay-to-win
