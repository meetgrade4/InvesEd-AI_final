## 0. Global Rules (Apply Everywhere)

- Brand colours: Primary `#1E3A5F`, Accent `#2E86AB`, Background `#F0F4F8`, Success `#1B6B3A`
- Every page has the `<Navbar />` fixed at top + `<AICoachFAB />` floating bottom-right
- Every XP gain triggers: `toast.custom(<XPToast amount={n} />)` with upward float animation
- Every badge unlock triggers: `<BadgeRevealModal />` full-screen overlay with Lottie animation
- Every level-up triggers: `<LevelUpModal />` with confetti Lottie burst
- Currency: always `₹X,XX,XXX` — use `formatINR(n)` util
- All charts use Recharts. All animations use Framer Motion.
- Mobile-first. Test at 375px width first.

**Question card (Framer Motion slide-in from right):**

- 4 answer options as large tappable cards
- Selected option animates to highlighted state
- "Next" button only appears after selection

**12 Questions — exact content:**

*Spending habits (Q1–Q4):*
1. "You get ₹2,000 pocket money. What do you usually do?" → [Save all / Save some, spend rest / Spend most / Spend all]
2. "You see a new game for ₹800 on sale for today only. You have ₹1,000 saved." → [Buy immediately / Think for 2 days / Skip, save the money / Ask parents]
3. "Your friend asks you to split a ₹500 Zomato order. You have ₹400 left this month." → [Pay it, figure it out / Decline politely / Ask to split ₹250 each / Charge it to next month's money]
4. "If you had ₹10,000, how long would it take you to spend it?" → [Same week / Same month / 3–6 months / Would invest it]

*Risk tolerance (Q5–Q8):*
5. "You invested ₹5,000 virtually. It drops to ₹4,000 in a week. What do you do?" → [Sell everything / Sell half / Hold and wait / Buy more]
6. "Which sounds more exciting?" → [Safe FD at 7% guaranteed / Mutual fund — might give 0% to 25% / Stock — might give -30% to 100% / Not interested in any]
7. "You flip a coin. Heads = win ₹1,000. Tails = lose ₹500. Do you play?" → [Yes / No / Only if I can afford to lose / Depends on my mood]
8. "Your investment is down 25% after 3 months. What do you tell yourself?" → [I knew this was risky / This will recover / I should have kept it in FD / I need to rethink everything]

*Financial goals (Q9–Q10):*
9. "Why do you want to learn investing?" → [Save for something specific / Build wealth long-term / Understand how money works / Beat friends on a leaderboard]
10. "What's your rough investment horizon?" → [Less than 1 year / 1–3 years / 3–7 years / 10+ years]

*Knowledge baseline (Q11–Q12):*
11. "What is a mutual fund?" → [A savings account / A pool of money invested in stocks/bonds / A type of bank loan / I don't know]
12. "What is a SIP?" → [A loan repayment / Investing a fixed amount monthly / A type of insurance / I don't know]

**Scoring:** Weighted sum → 0–40 = Conservative, 41–70 = Moderate, 71–100 = Aggressive
